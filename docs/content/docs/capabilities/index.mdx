# Capabilities

Galya Reality exposes a small, shared surface in Python and TypeScript:

1. Resolve a catalog validator by name
2. Stream context with **`index()`**
3. Score with **`judge()`** and read a **ValidationResult**

| Page | What it covers |
|------|----------------|
| [API model](/docs/capabilities/api-model) | Types, `validator()`, client, errors |
| [index()](/docs/capabilities/indexing) | Feeding context as messages arrive |
| [judge()](/docs/capabilities/judge) | Scoring and result fields |

Same semantics in both languages — only packaging and async entrypoints differ.
