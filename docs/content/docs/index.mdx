**Galya Reality** is the SDK for installing and running judgement validators in
Python or TypeScript. Call a validator by name, feed it context, and get a
score — with optional explanations when you need them.

![Self-improving Judgement Loop](/judgement-loop.jpg)

*Better agent judgement — move from instinct to reality grounded intuition for clearer insights.*

## What you get

- **One API** — `validator(name)` → `.index()` / `.judge()`
- **Pinned installs** — validators resolve from a shared catalog at a fixed commit
- **Optional explanations** — `judge()` can return human-readable reasons next to the score
- **Python & TypeScript** — same semantics in both languages
